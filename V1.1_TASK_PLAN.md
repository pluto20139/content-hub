## 任务 3：Edge Function `parse-url` 扩展

**目标**：支持知乎、抖音、小红书 URL 解析，返回 `native_type`，明确短链缓存与代理方案。

**子任务**：
- [ ] 知乎：
  - 支持 `www.zhihu.com/people/:id` → `native_type=people`
  - 支持 `www.zhihu.com/column/:id` → `native_type=column`
  - 支持 `zhuanlan.zhihu.com/c/:id` → `native_type=column`（若 RSSHub 支持）
  - **明确不支持** `zhuanlan.zhihu.com/p/:id` 单篇文章作为 monitor
- [ ] 抖音：
  - 支持 `v.douyin.com/:short` 短链重定向 → `sec_uid`
  - 支持 `www.douyin.com/user/:sec_uid`
  - 支持 `www.douyin.com/(:shareId)` 分享短链
- [ ] 小红书：
  - 支持 `xiaohongshu.com/user/profile/:uid`
  - 支持笔记短链 `xhslink.com/:short` 仅作为输入兼容（解析出 uid 则保存，解析不出则报错）
  - 若任务 0 调研确认不存在用户主页短链，则在错误提示中引导用户粘贴主页长链
- [ ] 短链解析：
  - fetch timeout 统一 10 秒
  - 先查 `short_link_cache` 表；未命中则解析，结果写入缓存，TTL 24 小时
  - 解析失败返回 `INVALID_URL` 或 `RSSHUB_ERROR`
- [ ] 昵称获取：
  - 知乎：RSSHub 路由
  - 抖音/小红书：HTML 解析或 RSSHub，失败时按 1.0 SPEC 格式降级：`{平台中文名}_{native_id前8位}`
- [ ] Edge Function 代理：
  - 通过 Supabase Function Secrets 注入 `EDGE_PROXY_LIST`
  - 代理格式与 Cron 统一，但读取方式在代码中明确区分
- [ ] 返回结构增加 `native_type` 字段

**交付物**：`supabase/functions/parse-url/index.ts`

## 任务 7：Cron 调度器注册新平台

**目标**：让 Cron 主流程识别并调度 5 个平台，明确新平台状态流转与平台级短路暂停机制。

**子任务**：

## 关键决策说明

| 问题 | 决策 | 说明 |
|---|---|---|
| 新平台失败状态流转 | 与 1.0 SPEC 状态机保持一致 | 单次非平台级失败进入 `cookie_expired`（语义为"抓取异常"），连续失败 ≥3 次进入 `rate_limited` 并告警，成功抓取后恢复 `normal`。 |
| 小红书移动网页兜底 URL | 以 Task Plan 为准，使用 `www.xiaohongshu.com` | PRD 已同步调整为 `www.xiaohongshu.com/user/profile/:uid`，带移动 UA 请求。 |
| 抖音图集 content_type | 统一映射为 `video` | 避免新增 `image_set` 类型导致 Deep Link 表、数据库 CHECK、前端展示都要改；Deep Link `snssdk1128://aweme/detail/{native_id}` 对图集和视频通用，若实测 App 无法识别图集，由 Deep Link 超时 fallback 到 `original_url`。 |
| 代理配置存储位置 | `platform_configs` | 优先从数据库读取 `proxy_list`，支持热更新；环境变量 `*_PROXY_LIST` 作为初始 seed 来源。 |
| 代理认证信息脱敏 | `platform_configs_admin` 视图对 `proxy_list` 返回 NULL | 与 `cookie` 同等敏感级别，管理员前端不展示代理密码。 |

## 风险与前置条件

| 风险 | 影响 | 缓解措施 |
|---|---|---|
| 抖音抓取被封 | 抖音内容中断 | 代理池 + 自托管 RSSHub + 反爬暂停 |

| 匿名隐藏被滥用 | 内容被批量隐藏 | 按当前决策不拦截，上线后观察日志 |
| 小红书没有用户主页短链 | parse-url 短链逻辑无用 | 任务 0 调研确认后调整 |
| RSSHub 路由已失效 | 知乎/抖音/小红书无法抓取 | 任务 0 调研，必要时更换抓取方案 |
