- **前端层**：采用 Vite + React + TypeScript 构建 Admin 管理端与 H5 用户端，通过 pnpm workspace 共享类型定义。
- **后端层**：依托 Supabase Cloud (PostgreSQL + PostgREST) 提供数据持久化与 REST API，利用 Edge Functions (Deno) 处理 URL 解析与 B站扫码等轻量逻辑。
- **自动化引擎**：通过 GitHub Actions 每 30 分钟触发 Node.js 脚本，经由平台适配器（B站/YouTube/知乎）抓取内容并写入数据库。
- **安全与权限**：实施严格的 RLS 策略区分管理员（authenticated）与访客（anon），敏感密钥仅存于 GitHub Secrets 或 Supabase Vault。
