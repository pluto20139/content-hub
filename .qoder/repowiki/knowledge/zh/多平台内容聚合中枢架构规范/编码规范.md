- 数据库表名与字段采用蛇形命名法（snake_case），如 `monitors`, `is_active`。
- Edge Function 名称使用连字符命名法（kebab-case）以确保 URL 友好，如 `parse-url`。
- TypeScript 类型与接口采用帕斯卡命名法（PascalCase），如 `Monitor`, `ContentCard`。
- SQL 迁移脚本文件采用数字前缀加蛇形命名法，如 `001_create_monitors.sql`。
- 前后端共享类型定义位于 `packages/shared`，作为单一数据源（Single Source of Truth）。
