- **环境变量配置**：需在 Vercel 和 GitHub Secrets 中配置 `SUPABASE_URL`, `SUPABASE_ANON_KEY`, `SUPABASE_SERVICE_ROLE_KEY`, `YOUTUBE_API_KEY` 等。
- **RSSHub 部署**：需独立部署 RSSHub 实例并配置 `RSSHUB_API_KEY` 以支持知乎内容抓取。
- **B站授权**：通过 Admin 端调用 `bilibili-auth` Edge Function 进行扫码登录以获取并加密存储 Cookie。
- **定时任务**：依赖 `.github/workflows/cron-fetch.yml` 自动执行抓取脚本，无需本地手动运行 Cron。
